**Programowanie obiektowe 1 - Laboratorium**
## Zadanie 1

Napisz program w języku Cpp.
Program musi składać się z trzech plików:

- main.cpp - plik główny programu

- pojazd.cpp - plik z ciałem klasy pojazd

- pojazd.h - plik nagłówkowy klasy pojazd

Klasa Pojazd zawiera metody:

- konstruktor,

- wirtualny destruktor,

- metodę: getWheelNumber()

- metodę: setWheelNumber(int val);

Klasa Pojazd zawiera pola prywatne:

- int wheelNumber;
